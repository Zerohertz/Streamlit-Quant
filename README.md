### How to Use?

```bash
$ docker pull zerohertzkr/streamlit-quant
$ docker run --name ${CONTAINER_NAME} -p ${PORT}:8501 zerohertzkr/streamlit-quant
```

### How to Remove?

```bash
$ docker stop ${CONTAINER_NAME}
$ docker rm ${CONTAINER_NAME}
$ docker rmi zerohertzkr/streamlit-quant
```
